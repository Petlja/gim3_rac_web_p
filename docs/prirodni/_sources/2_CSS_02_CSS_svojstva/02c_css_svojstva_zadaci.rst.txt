.. questionnote::

    **Задатак 2.2.1**

    Направите *HTML* страну са неколико наслова и параграфа. 
    
    - Испробајте подешавање својстава ``margin`` и ``padding`` у стиловима параграфа. 
    - За све параграфе привремено поставите оквир, да бисте јасније видели дејство различитих подешавања својстава ``margin`` и ``padding``.
