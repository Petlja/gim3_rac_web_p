Веб интерфејс - задаци
======================

.. questionnote::

    **Задатак 4.5.1**

    Направите веб страну са уграђеним калкулатором. Калкулатор треба да има дугмад за цифре, децималну тачку, знаке операција, заграде и знак ``=``. Поред дугмади, у веб страни треба да постоји параграф који представља екран калкулатора. Овај прараграф је иницијално празан, а при сваком притиску на неко дугме (осим дугмета ``=``) параграфу се дописује знак са тог дугмета.

    У функцији која се покреће кликом на дугме ``=``, користити функцију ``eval``, која израчунава вредност израза (на пример, ``eval("(2+3)*3")`` враћа вредност 15).
    
    .. code--block:: javascript

            function jednako() {
              // дохватити параграф који представља екран калкулатора
              p.innerText = eval(p.innerText);
            }

.. questionnote::

    **Задатак 4.5.2**


    Направити веб страну која приказује једну слику. Испод слике треба да буде падајућа листа за одабир, у којој су ставке описи неких других слика. Када корисник одабере ставку, одговарајућа повезана *JavaScript* функција треба приказану слику да замени одговарајућом, на пример тако што као вредност атрибута ``src`` упише путању до нове слике.
